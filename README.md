# UAI USB Boot

Universal AI USB Bootable Image

## Description

Portable bootable USB image for deploying UAI platform with zero-configuration setup.

## Getting Started

### Prerequisites
- USB Drive (8GB+)
- Linux or Windows with dd/Rufus
- Internet connectivity

### Installation

```bash
git clone https://github.com/Pikkati/UAI-USB-BOOT.git
cd UAI-USB-BOOT
./build_image.sh
```

## Features

- Zero-configuration deployment
- Persistent configuration
- Built-in Docker Swarm
- Multi-node clustering
